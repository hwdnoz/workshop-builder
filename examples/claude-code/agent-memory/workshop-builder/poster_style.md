# SVG Poster Style

## Orientation and Dimensions

- **Always landscape** — `viewBox="0 0 2400 1800"`
- Never portrait
- Wide format supports image-heavy layouts

## Image Density Rule

- **Every content zone must have a large image placeholder**
- No pure text zones
- Image placeholders should be at least 40-50% of the zone's visual space
- Examples: `[IMAGE: Coffee plantation in Ethiopia]`, `[IMAGE: 19th century coffee house]`, `[IMAGE: Modern espresso equipment]`

## Text Density

- Short headlines only (5 words max per section)
- One stat or key phrase per zone
- No full sentences
- Keep text brief and scannable

## Visual Hierarchy

- Header zone: topic title + key visual element
- 3-4 content zones: each with headline + stat + image placeholder
- Footer: closing line + contact or source info
- Clear visual separation between zones

## Color and Design

- Use a cohesive color palette (3-5 colors)
- Flat design only (rectangles, circles, lines — no gradients or filters)
- High contrast for readability
- Images should feel prominent, not supplementary

## Tone

- Visual-first approach — the images are the main story
- Captions and text support the visuals, not the reverse
- No arrogant or condescending language
- Plain, descriptive labels only

## Layout Example

```
[HEADER: "The History of Coffee" + decorative element]

[ZONE 1]
Headline: "Origins in Ethiopia"
Image: [40% of zone]
Fact: "12th century trade routes"

[ZONE 2]
Headline: "Global Expansion"
Image: [40% of zone]
Fact: "Coffee reaches Europe by 1700s"

[ZONE 3]
Headline: "Industrial Era"
Image: [40% of zone]
Fact: "Commercial cultivation spreads globally"

[ZONE 4]
Headline: "Modern Day"
Image: [40% of zone]
Fact: "2+ billion cups consumed daily"

[FOOTER: "Learn more: [URL]"]
```
